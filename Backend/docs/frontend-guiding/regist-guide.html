Fungsionalitas yang dibutuhkan di sisi client-side.

  • Form validation real-time (cek email, password, etc.)
  • Password strength indicator.
  • File upload preview & validation (ukuran, tipe file).
  • Fungsionalitas Drag & drop file upload.
  • Menyimpan progres ke `localStorage` (jika user refresh).
  • Transisi antar step yang mulus (jika digabung dalam 1 halaman).

âœ¨ UX, Security & Validation

UX Enhancements

  • Validation Messages: Tampilkan pesan error/sukses inline dengan icon.
  • Loading States: Tombol submit menampilkan spinner dan text "Menyimpan...".
  • Responsive: Card padding lebih kecil di mobile, font size disesuaikan.
  • Micro-interactions: Efek 'lift' (translateY) saat hover tombol, border 'glow' saat input focus.

Security & Validation

  • Frontend: Validasi dasar (required, email format, file size) di JS untuk UX cepat.
  • Backend: Validasi WAJIB di Controller (Laravel Request) untuk keamanan. Jangan pernah percaya input user.
  • Error Handling: Tampilkan error dari backend (via Blade `@error`) dan highlight field yang salah (border merah).
